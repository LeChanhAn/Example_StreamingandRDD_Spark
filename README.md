# Example_StreamingandRDD_Spark
Dự án này bao gồm hai ví dụ minh họa sức mạnh của Apache Spark:

- ETL Pipeline: Sử dụng Spark Structured Streaming kết hợp với Kafka để xử lý dữ liệu theo thời gian thực (Incremental Batch).

- Data Processing: Sử dụng Spark RDD để giải quyết bài toán đếm từ (Word Count) trên cụm máy chủ (Cluster).
